| **Business Size**      | Solo practitioners to multi-location clinics                       |

## Search Intent Mapping

| Intent Type       | User Goal                       | Content Format                | Keywords Signals                |
| ----------------- | ------------------------------- | ----------------------------- | ------------------------------- |
| **Informational** | Learn how/what/why              | Guides, tutorials, explainers | "how to," "what is," "why"      |
| **Commercial**    | Compare options before deciding | Reviews, comparisons, lists   | "best," "top," "vs," "review"   |
| **Transactional** | Ready to take action            | Service pages, strong CTAs    | "buy," "hire," "get," "near me" |

## Topic Selection Rules

1. **Check existing posts** — Never duplicate a topic already covered
2. **Rotate categories** — Don't publish 3 posts in same category consecutively
3. **Seasonal relevance** — Consider current month/season
4. **Audience fit** — Match topic to target audience needs
5. **Actionable content** — Every post should have takeaways
6. **Cluster awareness** — Prioritize cluster posts that support existing pillars
7. **Pillar first** — Create pillar content before cluster posts in new topic areas
8. **Internal linking** — Every cluster post must link to its pillar
9. **E-E-A-T for YMYL** — Medical/aesthetics content requires experience, expertise, authoritativeness, and trustworthiness signals
